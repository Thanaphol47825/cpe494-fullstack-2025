# Static Serve

## Folder structure

In the static folder, you can add any files.

```
[your module]/
├── static/
    ├── demo.html --> url: http://localhost:8080/[your module]/demo.html
    └── js/
        └── demo.js --> url: http://localhost:8080/[your module]/js/demo.js
```
